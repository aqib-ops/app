# Project Setup

## Requirements
- Node.js `22.12.0` (use `.nvmrc`)
- npm

## Install
```bash
nvm use
npm install
```

## Environment
Create `.env` and set values if needed:

- `VITE_N8N_WEBHOOK_URL` for contact form submissions

## Run
```bash
npm run dev
```

## Quality Checks
```bash
npm run lint
npm run build
```

## Preview Production Build
```bash
npm run preview
```
